# KokontsevOS — Project Brief

## Миссия
Построить персональную AI-систему **second brain advisor** — интеллектуального помощника, который понимает, классифицирует и сохраняет мои мысли, события и идеи.

## Цель v0.1
Создать минимальный, но полнофункциональный прототип, который:
- **Принимает** текстовые и голосовые сообщения (v0.1: только API, UI позже)
- **Классифицирует** входящие данные в 6 типов:
  - **Задача (task)** — что-то, что нужно сделать
  - **Дневной лог (daily_log)** — факт, событие, наблюдение из жизни
  - **Решение (solution)** — ответ или подход к проблеме
  - **Идея (idea)** — творческая мысль, гипотеза, концепция
  - **Блокер (blocker)** — проблема, препятствие, риск
  - **Запрос на план (plan_request)** — просьба структурировать информацию
- **Сохраняет** данные в Postgres с полным контекстом
- **Возвращает** краткий полезный ответ (резюме, следующий шаг, вопрос)

## Первичный пользователь
Один пользователь (система персональная и приватная).

## Что входит в v0.1
✅ API для приема сообщений и классификации  
✅ Postgres/Supabase как источник данных  
✅ OpenAI API для классификации и генерации ответов  
✅ Базовая структура БД  
✅ Система безопасности (API key, приватность данных)  

## Что НЕ входит в v0.1
❌ Telegram-бот (будет в v0.2)  
❌ Notion-синхронизация (будет в v0.3)  
❌ Dashboard и UI для просмотра (будет в v0.2)  
❌ Голосовая обработка (будет в v0.2, пока текст или интеграция)  
❌ Память (контекст из прошлых сообщений, будет в v0.3)  
❌ Weekly review (будет в v0.4)  
❌ Экспорт в Notion (будет в v0.3)  
❌ Аналитика и графики  
❌ Теги, фильтры, поиск (базовые, но не приоритет)  

## Целевой стек
- **Backend**: Next.js + TypeScript + API Routes
- **Database**: Supabase/Postgres
- **AI**: OpenAI API (GPT-4 mini для классификации)
- **Frontend**: Minimal — только API (UI позже)
- **Deployment**: Vercel (Next.js) + Supabase

## Timeline
- **v0.1** (текущий спринт): API + классификация + БД
- **v0.2** (месяц 2): Telegram + простой dashboard
- **v0.3** (месяц 3): Память + Notion-синхронизация
- **v0.4** (месяц 4): Weekly review + расширенная аналитика

## Успех
v0.1 считается успешной, когда:
1. API принимает текст и классифицирует его
2. Данные корректно сохраняются в БД
3. Система возвращает релевантный ответ
4. Базовая безопасность работает
